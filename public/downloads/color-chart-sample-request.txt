AMERICAN SUPER PANEL
COLOR STYLE & SAMPLE REQUEST GUIDE

Use this guide to describe the color style you need so sales can recommend
available finish options, samples, and alternates for the project.

Important:
The color names below are style references, not a promise of current stock.
Final color, coating system, substrate, gauge, trim match, warranty path, and
availability must be confirmed by project before release.

Common color style paths:

Metallic utility
- Examples: Galvalume, galvanized.
- Useful for shops, warehouses, agricultural buildings, storage buildings, and
  practical commercial packages.

Primer-ready / industrial design
- Example: Rezibond-style primer-ready.
- Useful when the buyer wants a weathered zinc / gunmetal industrial design
  direction.
- Confirm primer-ready availability, substrate, and final use before release.

Light cool-roof direction
- Examples: Polar White, light gray, selected reflective colors.
- Useful for heat-aware projects and light-color roof conversations.

Neutral painted direction
- Examples: Light Stone, Slate Gray, Charcoal.
- Useful for shops, commercial buildings, storage, trim, and siding packages.

Matte architectural direction
- Examples: Matte Black, low-glare dark colors.
- Useful for modern shops, retail, office, multifamily, and accent work.

Earth-tone direction
- Examples: Burnished Slate, Rustic Red, Forest Green.
- Useful for agricultural, ranch, shop, and storage buildings.

Specialty accent direction
- Examples: Copper Penny, wood grain, custom color coordination.
- Useful for trim, fascia, soffit, accent panels, and high-visibility details.

Sample request questions:
- Project name and state
- Roof, wall, trim, flashing, fascia, soffit, or accessory use
- Desired panel profile
- Desired color direction
- Finish priority: cost, cool roof, matte appearance, premium appearance,
  primer-ready design, or match existing building
- Gauge or substrate preference, if known
- Owner, architect, contractor, or dealer approval needs
- Timeline for sample review
- Alternate colors acceptable if preferred color is unavailable

Recommended next step:
Send project requirements, preferred color style, and any photos or drawings.
Sales can then recommend available samples and finish paths for review.
