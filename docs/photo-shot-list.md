# American Super Panel Photo Shot List

Professional photography is not required for launch or early replacement.

## Must-Have Photos

- Steel coils
- Machine running
- Panels exiting machine
- Trim fabrication
- Finished bundles
- Truck loading
- Warehouse interior
- Employees operating equipment
- Commercial project
- Agricultural project

## Phone Setup

- Use the newest iPhone available.
- Keep Portrait mode OFF.
- Keep HDR ON.
- Shoot landscape orientation.
- Take 10-20 shots per subject.

## Timing

- Shoot in morning light when possible.
- Shoot again in late afternoon light.
- Avoid harsh noon shadows, blown-out highlights, and heavy backlight.

## Composition Notes

- Favor real equipment, coils, panels, bundles, trucks, and work surfaces.
- Show enough context to feel like a real manufacturer.
- Avoid smiling hardhat stock poses, handshakes, fake AI workers, and overly polished marketing setups.
- Capture clean horizontal shots that can replace the current website image slots without layout changes.

## Replacement Workflow

1. Pick the best real photo for the matching key in `content/image-plan.md`.
2. Crop/export for web at roughly 1600-2000px wide.
3. Add the file to the future image asset location.
4. Update only `src/content/images.ts`.
5. Run `npm run build`.
6. Review the affected page at desktop and mobile sizes.
